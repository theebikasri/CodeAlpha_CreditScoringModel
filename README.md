# Credit Risk Prediction System

## Objective
Predict whether a loan applicant is a high-risk or low-risk borrower using Machine Learning.

## Algorithm Used
Random Forest Classifier

## Dataset
Credit Risk Dataset from Kaggle

## Features Used
- Person Age
- Annual Income
- Home Ownership
- Employment Length
- Loan Intent
- Loan Grade
- Loan Amount
- Interest Rate
- Loan Percent Income
- Previous Default Status
- Credit History Length

## Technologies
- Python
- Pandas
- Scikit-learn
- Streamlit

## Run Training

python credit_scoring.py

## Run Application

streamlit run app.py
